## 📖 项目介绍
### 项目用途
实现安卓手机自动化控制，通过脚本模拟点击、滑动、输入等操作，解放重复手工操作。

### 技术栈
- 核心脚本：JavaScript / Rhino
- 跨端逻辑：UTS
- 页面视图：HTML / CSS / SCSS

### 核心功能规划
- 设备模拟触控、手势自动化
